Integration Instructions

  1. Add the script: Include <script src="share-utils.js"></script> in your HTML head or before closing body tag.
  2. Create a container: Add a div with an ID where you want the share buttons to appear: <div id="shareButtons"></div>
  3. Initialize sharing: Create a new HowManyQShare instance and call createShareButtons().
  4. Customize: Use the configuration options to customize appearance and behavior.

Basic Usage Example:

// Basic usage
const sharer = new HowManyQShare({
    text: 'Check out my calculation result!',
    platforms: ['twitter', 'facebook', 'copy']
});

sharer.createShareButtons('shareButtons');

Advanced Usage Example:

// Advanced usage with custom callbacks
const sharer = new HowManyQShare({
    title: 'My Calculator Result',
    text: 'I just calculated something amazing!',
    hashtags: ['Calculator', 'Math'],
    platforms: ['native', 'twitter', 'facebook', 'linkedin', 'copy'],
    onShare: function(platform, options) {
        console.log('Shared on:', platform);
        // Track analytics, etc.
    },
    showToast: true,
    toastDuration: 3000
});

sharer.createShareButtons('shareButtons');
